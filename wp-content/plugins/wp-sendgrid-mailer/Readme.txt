WP SendGrid allows you to set your default WordPress emails through SendGrid

== Description ==
WP SendGrid plugin enables easy to do one step configuration to switch all your WordPress emails via SendGrid or SMTP. Now you can send WordPress email using SendGrid API using our SendGrid WordPress Plugin

= Highlights =
* Sending mail via SendGrid or SMTP
* Mail Logs can be accessed from Wordpress.
* Can verify campaign status like open, clicked, etc log view.

== Installation ==

1. Extract the SendGrid.zip in wordpress/wp-content/plugins using FTP or through plugin install in wp-admin.
2. Activate the plugin in the WordPress plugin list.
